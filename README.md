# Al-Qur'an Digital 📖

Aplikasi React untuk membaca Al-Qur'an per ayat, lengkap dengan teks Arab, Latin, dan terjemahan Bahasa Indonesia.

**Data API:** [equran.id](https://equran.id)

## Fitur

- 📖 114 **surat** lengkap
- 🅰️ Teks **Arab**, **Latin**, & **Terjemahan Indonesia**
- ⬅️ **Prev/Next** ayat (keyboard ⬅️ ➡️)
- 🔢 **Lompat** ke ayat tertentu
- 📊 **Progress bar** posisi bacaan
- 🌙 Tampilan gelap nyaman di mata

## Cara Pakai

```bash
npm install
npm run dev
```

Buka `http://localhost:5173` di browser.

## Build Produksi

```bash
npm run build
```

Hasil build di folder `dist/`.
